Tekniikasta

Sovellus on koodattu Reactilla tekoälyä käyttäen ja se on PWA (Progressive Web Application) sovellus. Koodit löytyvät osoitteesta https://github.com/SeijaLauronen/ostokseni
ja linkistä: Ostokseni sovelluksen koodit

Sovellus on vielä keskeneräinen, mutta sitä voi jo käyttää. Periaatteessa tiedot säilyvät selaimen muistissa, mutta kannattaa käyttää varmuuskopiointi ominaisuutta, koska kaikkia tilanteita ei vielä ole otettu huomioon, esim quotan ylittymistä, jolloin tiedot saattavat hävitä yllättäen.

About the Technology

The application is coded with React using artificial intelligence and is a PWA (Progressive Web Application). The code can be found at https://github.com/SeijaLauronen/ostokseni target="new">https://github.com/SeijaLauronen/ostokseni.
and the link: Code for the Ostokseni application

The application is still a work in progress, but it is already usable. In principle, data is retained in the browser's memory, but it's a good idea to use the backup feature, as not all situations have been accounted for yet, such as quota limits, where data may unexpectedly disappear.
